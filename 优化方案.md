1️⃣ 视觉风格大改造：注入“牛马班味”
你现在是白底黑字像后台管理系统，我们要把它变成暗黑荒诞打工人风格。

整体色调：背景 UI 用 深褐色/牛皮纸色（#2b1a10），搭配 荧光绿（#b3ff00，代表摸鱼值）和 警示橙（#ff6600）。

字体：去系统默认字体，改用 font-family: 'Courier New', monospace 加上 text-shadow，或者引入一个免费的像素字体（比如 Press Start 2P），立刻有内味。

卡片栏：现在的方框太硬了。改成 圆角+粗糙手绘边框（border: 3px dashed #ffcc00），卡片上加上 Emoji 图标（🌻、📄、🧱），文字换成“🌻 向日葵社畜”。

2️⃣ 布局排版优化（对标原版 PvZ）
现在的 UI 可能挤在一起，建议用 Flex/Grid 固定分区：

左上角：工位血量（改成红色血条样式，不要纯数字）+ 摸鱼值（大号荧光绿数字）。

右上角：下班倒计时（加大加粗，快结束时变红闪烁）。

顶部中央：事件弹窗/滚动字幕（“甲方改需求了！”这里做一个打字机效果，或加个闪烁的牛头 🐮 头像，显得很聒噪）。

底部：卡片栏 固定高度，选中的卡片放大且有发光边框。

右下角：“摸鱼锤”技能按钮做大一点，带冷却遮罩（CD转圈）。

3️⃣ 反馈特效升级（让操作“爽”起来）
现在的点击和事件反馈可能比较“干”。

种植反馈：在 3D 网格上点击时，立即生成一个绿色的半透明方块预览（跟随鼠标），放下时播放“尘土飞扬”的粒子爆炸效果（Three.js Points）。

摸鱼值变动：获得 +25 时，从向日葵头顶飘出一个带“+25 🐟”的 3D 文字或 CSS 浮动标签，向上飘散消失。

倒计时压迫感：最后 60 秒，整个 UI 边框变成红色脉冲，3D 场景主光源变暗闪烁（模拟加班深夜）。

4️⃣ 3D 场景内部优化
光照：你现在可能只有环境光。加上 方向光 + 聚光灯，让植物和僵尸有立体感（投影不用开太高性能，但要有明暗对比）。

地面：5x9 网格加上淡灰色辅助线，鼠标悬停的格子高亮为黄绿色。

相机：略微调整角度，用 45度正交视角（OrthographicCamera）会更像原版塔防，如果喜欢透视，把相机拉高一点（position.set(12, 18, 12)）。
