Three.js 开发提示词：《植物大战僵尸：牛马版》
1. 角色与目标
你是一名资深 Three.js 游戏开发专家。请使用 Three.js (r152+) 配合 Vite 构建工具，开发一款名为《植物大战僵尸：牛马版》的3D塔防游戏原型。核心原则：用“打工人荒诞感”重构经典玩法，强调随机事件干扰与生存压力。

2. 技术栈与环境
渲染：Three.js (透视相机 PerspectiveCamera，俯视45度角)。

交互：Raycaster 实现鼠标点击地块选择与卡片拖拽放置。

UI层：使用 HTML/CSS 叠加层 绘制血条、阳光/摸鱼值、卡片栏和对话气泡（避免3D文字复杂渲染）。

状态管理：使用原生 EventEmitter 或简单单例模式管理游戏状态（资源、波次、事件）。

构建：Vite + 原生ES Module。

3. 核心游戏机制 (需编码实现)
3.1 场景与网格系统
建立 5行 x 9列 的棋盘格，每个格子尺寸为 2 x 2 单位。

使用 BoxGeometry 或 PlaneGeometry 绘制草地格子，行间用不同深浅绿色区分。

左侧为“房区”（僵尸生成点），右侧为“工位”（玩家基地），基地血量用顶部UI条展示。

3.2 资源系统（“摸鱼值”代替阳光）
初始资源 50点摸鱼值。

点击“向日葵社畜”卡片消耗 50点，种植后每5秒生产 25点摸鱼值（飘出“+25”3D浮动文字）。

顶部UI显示当前摸鱼值，数字变化带缓动动画。

3.3 “牛马”植物单位（使用简单几何体组合 + 精灵图纹理占位）
向日葵社畜：黄色圆球+绿色茎，产出摸鱼值，附带“抖腿”待机动画。

PPT豌豆射手：深蓝色方块+发射管，发射白色“Word文档”弹丸（SphereGeometry 带贴图），伤害20，攻速1s。

996坚果墙：灰色厚重长方体，血量800，被攻击时表面出现“裂纹”效果（替换材质或缩放顶点），附带“黑眼圈”贴花。

摸鱼锤（主动技能）：点击锤子卡片后，光标变为锤子，点击地块上的“牛头指导”气泡可敲碎它（打断负面事件）。

3.4 “魔性”僵尸单位（沿直线向左移动）
甲方僵尸：粉色模型，移动速度慢，但每走3步会“突然变向/后退”（模拟需求变更）。

老板僵尸：金色模型，自带光环（RingGeometry 旋转），经过的植物攻击力减半（光环范围2单位）。

KPI僵尸：红色模型，如果它在场上存活超过10秒，会扣除玩家基地血量（模拟绩效扣款）。

3.5 🐮 “牛头指导”干扰系统（核心特色）
触发：游戏开始后每 15-25秒随机触发。

展现：屏幕中央弹出HTML模态框，显示“牛头”头像和两条选项（例如：A. 给所有植物加攻但扣30摸鱼值；B. 获得50摸鱼值但随机冻结一块植物5秒）。

实现：选择后立即执行对应的 buff/debuff 函数。若玩家未在 8秒内 选择，系统将“强制帮你选最坑的那项”。

打断机制：若玩家手速快，可使用“摸鱼锤”点击场景中随机出现的“牛头幽灵”3D物体，直接取消本次干扰（但有20%概率触发“反向加班”惩罚，倒扣资源）。

3.6 胜利与失败条件
胜利：坚持 5分钟（游戏内倒计时），出现“带薪拉屎成功！”庆祝画面。

失败：基地血量归零，或“摸鱼值”变为负数（发薪日扣款不足），出现“提桶跑路”灰屏画面。

4. 视觉与动画风格要求
建模：所有单位使用 低多边形 (Low Poly) 风格（BoxGeometry, CylinderGeometry, ConeGeometry 组合），拒绝光滑材质，使用 MeshLambertMaterial 加纯色。

动画：待机时做 Y轴上下浮动 和 Z轴小幅度旋转（用 requestAnimationFrame 驱动）。

粒子效果：植物种植时播放绿色圆环扩散粒子（Points 系统），僵尸死亡时播放灰色方块爆裂粒子。

// main.js 骨架
import * as THREE from 'three';
import { OrbitControls } from 'three/addons/controls/OrbitControls.js';

const scene = new THREE.Scene();
scene.background = new THREE.Color(0x87CEEB); // 天空蓝
const camera = new THREE.PerspectiveCamera(45, window.innerWidth/window.innerHeight, 0.1, 100);
camera.position.set(15, 20, 15);
camera.lookAt(0, 0, 0);

const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(window.innerWidth, window.innerHeight);
document.body.appendChild(renderer.domElement);

// 添加基础光照
const ambientLight = new THREE.AmbientLight(0x404060);
scene.add(ambientLight);
const dirLight = new THREE.DirectionalLight(0xffffff, 1);
dirLight.position.set(10, 20, 5);
scene.add(dirLight);
